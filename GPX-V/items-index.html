GPX-V Main Window

  • Action Buttons perform available actions. Grouped by video, video and GPS, and GPS.
  • Add Trackpoint Button adds a trackpoint at the location on the map at the time specified in video
  • Delete Trackpoints Button deletes the selected trackpoints
  • Directional Synch Menu controls the direction of immediate synchronization of video and trackpoints
  • Elevation Button Adjusts elevations and gradients
  • Elevation Chart shows how your route changes elevation. Colors signal the gradient of the slope. Click to select a point along the route. Shift-click to select a range from the current selected point.
  • Export GPX Button exports a synchronized GPX file
  • Export Video Button exports a syncrhonized .mp4 file
  • GPX-V Document Window displays video, map, elevation chart, trackpoint table, and document contents.
  • Import GPX Button imports a GPX file
  • Import Video Button imports one or more .mp4 video files
  • Infer Checkbox when checked, inferred trackpoints are displayed
  • Information Option Control displays the three classes of available information. Click to select the one you want to see.
  • Information View shows information about the history of changes to your document, an analysis of any potential problems, and the current status of your document.
  • Map Type Control switches between traffic and satellite views.
  • Remove Audio Button deletes audio tracks
  • Route Interpolation Button interpolates a route between two selected points
  • Route Map displays your route.
  • Segment Delete Button deletes selected trackpoints and video
  • Selected Video Frames shows 1, 2, or 3 frames depending on how long the video selection is. First, middle, and last frames of the selection.
  • Selection Range Slider can change the selected video and trackpoints. Slide, click or shift-click to change the selection.
  • Share Video Button shares a frame or clip
  • Speed Adjustment Button opens an editor to apply speed adjustments to video and GPX simultaneously.
  • Speed Change Control shows any changes in speed that have been applied to the route. Click a filter to edit.
  • Titles Button opens an editor to add text to the video for a selected segment
  • Trackpoint Edit Button Edits the selected trackpoints
  • Trackpoint Table lists the GPS trackpoints in order. Click a row to select the point, shift-click to select a range. Arrow keys work too.
  • Trim From Start Button trims from the start of the video until the first selected trackpoint
  • Trim Until End Button trims from the last selected trackpoint until the end of the video
  • Video Filter Button opens an editor to apply image filters to the selected video to compensate for shadows, glare, etc.
  • Video Filter Display shows the application of video filters to the video. Click a filter to edit.
  • Video Player plays the imported and merged video files.

Title Editor

  • Add Button adds a new title to the list.
  • Apply/Update button saves any changes you made and quits the Title Editor.
  • Cancel button discards the changes you made and quits the Title Editor.
  • Delete Button deletes the selected title.
  • Font Controls set the font family, style, size and color that will be applied.
  • Frame Preview shows the first, middle, and last video frames to which the title will be applied.
  • Preview button opens a window that plays a preview of the selected title.
  • Range Label shows the start, finish and length of the video segment to which the title will be applied.
  • Range Slider selects the time range in the video to which the selected title will be applied.
  • Subtitle shows the subtitle that will be applied. Click to edit.
  • Table View lists the titles you have applied to the video.
  • Text Placement Control positions the title text in the video frame. Click to select the desired position.
  • Title shows the title that will be applied. Click to edit.
  • Title Editor inserts and formats text into segments of the video.

Video Filter Editor

  • Add Button adds a new filter to the list.
  • Apply/Update button saves any changes you made and quits the Video Filter Editor.
  • Cancel button discards the changes you made and quits the Video Filter Editor.
  • Color Temperature Slider adjusts the color temperature of the applied filter.
  • Color Tint Slider adjusts the tint of the applied filter.
  • Delete Button deletes the selected filter.
  • Gamma Slider adjusts the gamma (mid-tone lift) of the applied filter.
  • Highlight Protection Slider adjusts the highlight protection of the applied filter.
  • Original Frame View shows the first, middle, and last video frames to which the filter will be applied.
  • Preview button opens a window that plays a preview of the applied filter.
  • Preview Frame View shows the effect of the applied filter on the first, middle, and last video frames.
  • Range Label shows the start, finish and length of the video segment to which the filter will be applied.
  • Range Slider selects the time range in the video to which the selected filter will be applied.
  • Shadow Lift Slider adjusts the shadow lift of the applied filter.
  • Table View lists the filters you have applied to the video.
  • Video Filter Editor defines and applies video filters to segments of the video.

Interpolation Editor

  • Apply Button inserts the interpolated trackpoints into your route.
  • Interpolation Type Control chooses between Bicycle and Automobile routing using Apple Maps, or plain linear interpolation.
  • Interpolation Type Control chooses between Bicycle and Automobile routing using Apple Maps, or plain linear interpolation.
  • Map View shows the existing route, the selected segment, and the expected interpolation.
  • Time Range Label shows the video and trackpoint time interval you have selected.
  • Trackpoint Interpolation Editor improves the route accuracy between two waypoints.

GPX-V Library

  • Clear Button clears the project list. For DEBUG purposes.
  • GPX-V Library Window displays all the projects you have created, across devices and volume.
  • Library Summary displays the total number of your projects, source video storage used, and distance covered.
  • Machine Popup controls the devices from which you are displaying GPX-V projects in the Project Table.
  • Notes Label shows additional information as needed.
  • Open Button opens the selected project. You can also double-click the project in the table.
  • Project Table displays projects and their associated data items.
  • Refresh Button updates the list of projects.
  • Scan Button searches for projects in the folder you select.
  • Volume Popup controls the volumes from which you are displaying GPX-V projects in the Project Table.
